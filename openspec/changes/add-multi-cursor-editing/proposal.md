# 多光标编辑功能提案

## Why（为什么）

多光标编辑是现代代码编辑器的核心功能之一，可以显著提升开发效率。用户经常需要在多个位置同时进行相同的编辑操作（如批量重命名变量、添加相同的代码模式等）。虽然 Monaco Editor 内核已支持多光标，但目前缺少明确的 UI 提示和用户指引，导致用户可能不知道如何使用该功能。

## What Changes（改变什么）

- 启用 Monaco Editor 的多光标编辑功能
- 添加多光标快捷键到欢迎页面的快捷键提示列表
- 在设置中添加多光标相关的配置选项
- 更新国际化文案，支持中英文说明
- 添加用户文档，说明多光标的使用方法

**核心快捷键：**
- `Cmd/Ctrl + Click` - 添加光标
- `Cmd/Ctrl + Alt + ↑/↓` - 在上方/下方添加光标
- `Cmd/Ctrl + D` - 选中下一个相同内容
- `Cmd/Ctrl + Shift + L` - 选中所有相同内容
- `Esc` - 取消多光标，回到单光标

## Impact（影响范围）

- **editor** - 编辑器核心功能（新增规格）

- `src/i18n/locales/zh.json` - 中文国际化文案
